# 五子棋 (Gobang)

一个精美的网页版五子棋游戏，支持人人对战。

## 功能特性

- 🎮 黑白双方轮流落子对弈
- 🏆 率先连成五子获胜
- ↩️ 支持悔棋功能
- 🔄 重新开始游戏
- 👤 用户登录系统（邮箱登录）
- 💾 登录状态持久化

## 快速开始

```bash
# 在浏览器中打开
open core/index.html
```

## 项目结构

```
gobang/
├── core/           # 核心游戏代码
│   ├── index.html  # 主页面
│   ├── game.js     # 游戏逻辑
│   ├── style.css   # 样式文件
│   ├── auth.js     # 认证服务
│   └── auth-ui.js  # 登录界面组件
├── docs/           # 文档
├── tests/          # 测试文件
└── hooks/          # Git hooks
```
