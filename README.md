# 🧱 网页五子棋 (Gobang)

一个基于 HTML5 + CSS3 + JavaScript 的网页版五子棋游戏。

## ✨ 功能特点

- 🎮 15x15 标准棋盘
- 🤖 双人对战模式
- 🏆 自动胜负判定
- ↩️ 悔棋功能
- 🔄 重新开始
- 📱 响应式设计
- 🎨 美观的 UI 界面

## 🚀 快速开始

### 方法 1：直接打开

1. 克隆或下载本项目
2. 双击打开 `index.html`
3. 开始游戏！

### 方法 2：本地服务器

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 或使用 PHP
php -S localhost:8000
```

然后访问 http://localhost:8000

## 🎮 游戏规则

1. 黑白双方轮流落子
2. 率先连成五子的一方获胜
3. 横、竖、斜向均可连子

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计（渐变、阴影、动画）
- **JavaScript (ES6+)** - 游戏逻辑
- **Canvas API** - 棋盘和棋子绘制
