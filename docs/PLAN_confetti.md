# 撒花动画功能计划

## 任务概述
为五子棋游戏增加获胜时的撒花动画效果

## 技术方案
- 使用 DOM + CSS 动画实现（轻量、流畅）
- 创建独立的 `confetti.js` 模块
- 在 `showWinner()` 中触发动画
